ENCODE Registered Experimental Variables

To facilitate integration of data between the contributing research groups during the first production phase (September 2007 - July 2012), the ENCODE Consortium implemented a controlled vocabulary. This page lists the registered experimental variables that make up this vocabulary.

These three registered experimental variables have their own pages, which also have direct links from the ENCODE at UCSC home page:

  • Antibodies
  • Human Cell Types
  • Mouse Cell Types

Each of the following registered experimental variables has a section on this page:

  • Age

  • Map Algorithms
  • Mouse Strains
  • Phases
  • Promoters
  • Protocols
  • Read Types
  • RNA Extracts
  • Sequence Platforms
  • Sex
  • Treatments
