Administrative Links for the UCSC Genome Browser

The administrative pages contain information on the following topics:

  • Creating a Genome Browser Mirror Site: Step-by-step instructions for incrementally creating a full or partial mirror browser.
  • hgFindSpec How-To: Technical reference for the hgFindSpec program (and related programs), the mechanism for adding searches to a trackDb.ra file.
  • Web Server Statistics: Several reports showing web server statistics for genome.ucsc.edu.
