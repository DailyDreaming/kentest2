Description

This column shows that best blastp match to the known genes protein set from the UCSC Human Genome Browser database.

Action

Click on an accession number to display the gene in the UCSC Human Genome Browser.

Method

The NCBI blastall program was run with the following parameters:
 -p blastp -e 0.001 -m 8 -b 1
