Description

This column shows the Affymetrix ID from the RAE230A and RAE230B chips that best corresponds to each gene.

Methods

Target genome sequences were downloaded from the Affymetrix Product Support and mapped to the genome using blat followed by pslReps with the parameters -minCover=0.3 -minAli=0.95 and -nearTop=0.005. For each gene, the target with the most overlap (on the same strand and excluding introns) was chosen.
