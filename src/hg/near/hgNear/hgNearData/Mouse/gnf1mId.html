Description

This column shows the Affymetrix ID from the GNF1M chip that best corresponds to each gene.

Methods

Target genome sequences were obtained from the Novartis Research Foundation (GNF) and mapped to the genome using blat followed by pslReps with the parameters -minCover=0.30 -minAli=0.95. For each gene, the most overlapping target (on the same strand and excluding introns) was selected.

Credits

The data for this column was kindly provided by GNF as part of the GNF Gene Expression Atlas.
