Description

This column provides a link to the Allen Brain Atlas (ABA) images for each gene. ABA is an extensive database of high resolution in situ hybridization images of adult male mouse brains covering the majority of genes. The identifiers shown in this column correspond to the RNA probe.

Action

Click on an ABA ID to query the ABA for images related to that gene.
