Description

This column shows the accession number of the best blastp match to the known genes protein set in the UCSC Rat Genome Browser database.

Action

Click on the accession number to display the gene in the UCSC Rat Genome Browser.

Methods

The NCBI blastall program was run with the following parameters:
 -p blastp -e 0.001 -m 8 -b 1
