Description

This column shows the FlyBase ID of the best blastp match to the FlyBase protein set.

Action

Click on the FlyBase ID to display the corresponding FlyBase report.

Method

The NCBI blastall program was run with the following parameters: 
 -p blastp -e 0.01 -m 8 -b 1
In the most recent assemblies, the best blat matches were then pruned to retain only the reciprocal-best matches.
