Description

This column shows the percentage identity at the protein level between each gene and the selected gene.

Method

All known splice variants of all known $organism genes are put into a blast protein database using the formatdb program. This database is then searched with each known gene using version 2.2.5 of the NCBI blastall program with the following flags: 
 -p blastp -e 0.01 -m 8 -b 1000
These flags indicate that the program should do a protein-based search and report up to 1000 hits of e-value 0.01 or better in a tab-delimited format. The output of this search is loaded into the Gene Sorter database.
