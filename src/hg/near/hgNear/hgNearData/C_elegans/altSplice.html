Description

This column displays instances of alternatively-spliced exons detected by examining mRNA and EST alignments. Spliced instances are indicated by the word "exon".

Methods

The first pass was done with tools developed for the Intronerator. This was followed by a second, manual curation pass by members of the Zahler lab. The manual curation selected exons that are included in some isoforms but not others and for which there was solid evidence.
