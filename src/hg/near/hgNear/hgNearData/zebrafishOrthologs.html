Description

This column shows the Ensembl peptide ID of the best blastp match to the Ensembl gene predictions on the zebrafish genome.

Action

Click on the Ensembl peptide ID to display it in Ensembl Zebrafish Protein View.

Methods

The NCBI blastall program was run with the following parameters: 
 -p blastp -e 0.005 -m 8 -b 1
