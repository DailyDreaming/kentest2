Description

This column displays protein abundance information, as reported in Ghaemmaghami et al. (2003), "Global analysis of protein expression in yeast", Nature 425:737-741. This study used a Staph. aureus protein-A tag fused to each yeast ORF to measure the absolute expression levels of each protein by quantitative immunoblotting.

For more information, see the Yeast GFP Fusion Localization Database.
