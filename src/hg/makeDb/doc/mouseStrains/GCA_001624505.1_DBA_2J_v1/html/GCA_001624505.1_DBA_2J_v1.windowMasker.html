Description

This track depicts masked sequence as determined by WindowMasker on the the 26 Apr 2016 Mus musculus (house mouse)/GCA_001624505.1_DBA_2J_v1/GCA_001624505.1 genome assembly. The WindowMasker tool is included in the NCBI C++ toolkit. The source code for the entire toolkit is available from the NCBI FTP site.

Methods

To create this track, WindowMasker was run with the following parameters: 

windowmasker -mk_counts true -input GCA_001624505.1_DBA_2J_v1.unmasked.fa -output wm_counts
windowmasker -ustat wm_counts -sdust true -input GCA_001624505.1_DBA_2J_v1.unmasked.fa -output windowmasker.intervals
perl -wpe 'if (s/^>lcl\|(.*)\n$//) { $chr = $1; } \
   if (/^(\d+) - (\d+)/) { \
   $s=$1; $e=$2+1; s/(\d+) - (\d+)/$chr\t$s\t$e/; 
   }' windowmasker.intervals > windowmasker.sdust.bed
The windowmasker.sdust.bed included masking for areas of the assembly that are gap. The file was 'cleaned' to remove those areas of masking in gaps, leaving only the sequence masking. The final result covers 712,172,685 bases in the assembly size 2,606,162,158 for a percent coverage of % 27.33.
