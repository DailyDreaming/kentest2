Description

This track depicts masked sequence as determined by WindowMasker

Methods

WindowMasker was operated with the following parameters: 
windowmasker -mk_counts true -input oryLat1.fa -output wm_counts
windowmasker -ustat wm_counts -sdust true -input oryLat1.fa -output repeats.bed
The repeats.bed (bed3) file was loaded into the "repeats" table for this track.

References

Morgulis A, Gertz EM, Schäffer AA, Agarwala R. WindowMasker: window-based masker for sequenced genomes. Bioinformatics. 2006 Jan 15;22(2):134-41. Epub 2005 Nov 15.
