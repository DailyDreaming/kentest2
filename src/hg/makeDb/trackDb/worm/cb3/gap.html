Description

This track depicts gaps in the assembly.

Gaps are represented as black boxes in this track. All these gaps are arbitrary 100 base pair gaps between ultracontigs. The relative order and orientation of the ultracontigs is not known on this artifical chrUn assembly. None of these gaps are bridged.

This assembly contains the following principal types of gaps:

  • clone - gaps between the Whole Genome Shotgun ultracontigs
  • contig - gaps between the Whole Genome Shotgun ultracontigs
