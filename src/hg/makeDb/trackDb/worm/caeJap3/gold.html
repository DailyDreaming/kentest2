Description

This track shows the draft assembly of the $organism genome. The genome has been sequenced to 22X coverage. See also: ABLE03 WGS information.

In dense mode, this track depicts the contigs that make up the currently viewed contig.

All components within this track are of fragment type "D": Whole Genome Shotgun draft contig.

Assembly details

This genome has been sequenced to 22X coverage. There are 35,931 contigs comprising 154,057,934 nucleotides. The largest contig, ABLE03030821 is 151,170 bases. One half of the sequence, 77,028,967 bases, is contained in 3893 contigs, all of at least 10,166 bases (N50).
