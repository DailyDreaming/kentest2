Description

This track shows alignments of $o_organism to $organism using the Wobble Aware Bulk Aligner (WABA), an alignment tool developed by Jim Kent for doing large-scale alignments between genomic DNA of different species.

Methods

The waba alignment algorithm is described in Kent, WJ and Zahler, AM (2000), "Conservation, Regulation, Synteny, and Introns in a Large-scale C. briggsae-C. elegans Genomic Alignment", Genome Res 10(8):1115-1125.
