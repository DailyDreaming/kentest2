Description

This track contains mutations in the Spike protein that lead to escape from a single patient with highly neutralizing COVID-19 convalescent plasma.

Display Conventions and Configuration

There are only three mutations. Features are labeled with the nucleotide and protein coordinates and the name of the antibody. Click a feature or mouse-over a feature to show these annotations.

Methods

Data was manually added from the paper.

Data Access

The raw data can be explored interactively with the Table Browser, or combined with other datasets in the Data Integrator tool.
