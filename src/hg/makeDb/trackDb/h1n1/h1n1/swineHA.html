Description

This track shows a collection of the HA gene sequences of all swine H1N1 (including the swine flu) viruses downloaded from the GISAID EpiFlu Database on 4/26/09.

Methods

A set of all swine H1N1 DNA sequences were downloaded from the GISAID EpiFlu Database. These sequences are aligned to the H1N1 base genome using BLAT. The resulting high scoring (>500) alignments were used to create this track in PSL format.
