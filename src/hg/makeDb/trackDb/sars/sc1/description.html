The SARS coronavirus TOR2 draft genome assembly is based on sequence deposited into GenBank as of 14 April 2003.

Sample position queries

A genome position can be specified by the accession number of an mRNA, a coordinate range, a gene identifier, or keywords from the GenBank description of an mRNA. The following list shows examples of valid position queries for the SARS coronavirus TOR2 genome. See the User's Guide for more information.

Request:
    Genome Browser Response:

chr1 Displays the entire genome for SARS coronavirus TOR2
chr1:15720-186200 Displays the region between bases 15720 and 186200 showing the protein NP_828870.1.

AF391541 Displays region of mRNA with GenBank accession AF391541
CS000003 Displays region of genome with fgenesv+ gene identifier CS000003

bovine coronavirus Lists mRNAs associated with bovine coronavirus
Chouljenko,V. Lists mRNAs deposited by co-author V. Chouljenko

Use this last format for author queries. Although GenBank requires the search format Chouljenko V, internally it uses the format Chouljenko,V..

Assembly details

This browser — which represents a departure from UCSC's usual focus on vertebrate genomes — shows gene predictions, locations of putative proteins, and viral mRNA and protein alignments. Protein structure analysis and predictions were determined by using the Sequence Alignment and Modeling (SAM) T02 tool.

UCSC does not plan to provide a comprehensive set of browsers for viruses. However, this browser will be maintained as long as there is strong scientific and public interest in the SARS coronavirus TOR2.

The SARS sequence and annotation data may be downloaded from the UCSC Genome Browser FTP server or Downloads web page. The annotation tracks displayed in the Genome Browser were generated by UCSC and collaborators worldwide.
