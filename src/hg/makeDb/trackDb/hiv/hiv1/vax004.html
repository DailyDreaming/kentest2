Description

This track shows VAX004 HIV-1 gp120 nucleotide sequences aligned against the $organism genome.

To choose a set of subjects to view in this track, visit the Table View. If there are no subjects chosen, this track will display a default set of subjects.

Methods

VAX004 HIV-1 gp120 nucleotide sequences were aligned to the base genome using BLAT. The resulting alignments with minimum identity of 80% and BLAT score equal or greater than 200 were used to construct this track.
