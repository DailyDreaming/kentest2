Sample position queries

A genome position can be specified by the accession number of a sequenced genomic region, an mRNA or EST, a chromosomal coordinate range, or keywords from the GenBank description of an mRNA. The following list shows examples of valid position queries for the lizard genome. See the User's Guide for more information.

Request:   Genome Browser Response:
 
chrLGh   Displays all of microchromosome chrLGh
chrLGh:1-1000000   Displays first million bases of microchromosome chrLGh
chrLGh:100000+2000 Displays a region of microchromosome chrLGh that spans 2000 bases, starting with position 100000
 
AF223224   Displays region of mRNA with GenBank accession number AF223224
AF508304   Displays human mRNA with GenBank accession AF508304 aligning to anoCar2 assembly
 
androgen   Anolis carolinensis androgen receptor mRNA, partial cds
homeobox caudal   Lists mRNAs for caudal homeobox genes
zinc finger   Lists many zinc finger mRNAs
kruppel zinc finger   Lists only kruppel-like zinc fingers
zahler   Lists mRNAs deposited by scientist named Zahler
Kitaguchi,T.   Lists mRNAs deposited by co-author T. Kitaguchi
 
Use this last format for author queries. Although GenBank requires the search format Kitaguchi T, internally it uses the format Kitaguchi,T..

Assembly details

The lizard genome has been sequenced to 7.1X coverage. The draft sequence was assembled into the chromosomes chr1, chr2, chr3, chr4, chr5, and chr6, as well as several microchromosome linkage groups that have not yet been assigned numbers: chrLGa, chrLGb, chrLGc, chrLGd, chrLGf, chrLGg, chrLGh. Names will be assigned to several of these microchromosomes in future releases; for example, chrLGc=chr7, chrLGd=chr8, chrLGa=chr9, and chrLGb=chrX. This assembly also contains 6,443 scaffolds with 717,481,773 bases of sequence that have not been localized to chromosome positions. The largest scaffold, chrUn_GL343193, comprises 14,352,415 bases. The scaffolds have an N50 size of 1,224,583; half of the additional sequence is contained in 154 scaffolds of at least this size. The total sequence size including chromosomes and scaffolds is 1,799,143,587 bases. The Genome Browser also displays the mitochondrion (chrM) sequence NC_010972 from GenBank. For more information on the AnoCar2.0 assembly, see the assembly record in GenBank.

Bulk downloads of the sequence and annotation data are available via the Genome Browser FTP server or the Downloads page. These data have specific conditions for use. The lizard browser annotation tracks were generated by UCSC and collaborators worldwide. See the Credits page for a detailed list of the organizations and individuals who contributed to this release.
