Description

This track shows areas of coverage for genomic regions of 3 alternate strains of chicken sequenced by the Beijing Genomics Institute (BGI): Broiler, Layer, and Silkie. Single reads from those three strains were mapped to the reference assembly for the Red Jungle Fowl (RJF).

Coverage area names are of the format [strain].[chrom].[start].[end]:

  • strain: Strain from which read was taken (Broiler, Layer or Silkie)
  • chrom: Chromosome in the RJF reference assembly to which read was mapped
  • start: First base in chrom of mapped read
  • end: Last base in chrom of mapped read
