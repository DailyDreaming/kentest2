Description

This track shows locations of likely contamination in the assembly, e.g. high-scoring alignments of E.coli, vector and maize mitochondrion sequence. The name of each feature indicates the type of contamination found there.

The maize mitochondrion contamination ("MAIZE_MITO") locations represent the entire whole genome shotgun (WGS) contig found to be contaminated. The E.coli and vector locations show only the contaminated portion of the affected WGS contig.
