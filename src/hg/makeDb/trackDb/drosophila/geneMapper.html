Description

This track shows gene predictions of GeneMapper, a program for transferring gene annotations (including UTRs) from a reference genome to newly sequenced genomes. These annotations have been obtained by transferring D. melanogaster RefSeq genes to the $Organism genome.

Methods

Whole genome homology maps of 9 fruitfly genomes were obtained using Colin Dewey's Mercator program. The homology map was used as a guide to transfer D. melanogaster RefSeq genes to the other genomes.

References

Chatterji S, Pachter L. Reference based annotation with GeneMapper. Genome Biol. 2006;7(4):R29. PMID: 16600017; PMC: PMC1557983
