Description

This track shows the alignment of the sequencing reads to the genome. BAM files were provided by the data submitters, one or two files per cell. All BAM files for the cells of a cluster were merged and are shown here together. The readgroup (RG) BAM tag indicates the original cell.

Display Conventions and Configuration

Alignable regions are shown in back, unalignable regions between two alignable ones shown with thin lines. For configuration options, see the BAM tracks help page

Methods

The alignments were used as submitted. They were merged with "samtools merge" into a single BAM file per cluster.

Data Access

The merged BAM files can be downloaded from the track hub directory.
