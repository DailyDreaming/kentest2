Description

This track displays alignments of the National Institute on Aging (NIA) Mouse Gene Index (mm8) against the mouse genome.

Methods

The index was assembled from Blat alignments of the mouse genome (mm8/Feb. 2006).

See the NIA Mouse cDNA Project home page for more information.
