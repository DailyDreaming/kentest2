Description

This track shows the draft assembly (Mar. 2016, UNIVERSITY OF WASHINGTON) of the $organism genome. This assembly used Pacific Biosciences (PacBio) long-read sequencing technology as described in the supplemental materials of: Long-read seqence assembly of the gorilla genome. Whole genome shotgun long (>10 kbp) sequence reads were assemblied using a string graph assembly algorithm, Falcon, and consensus algorithm, Quiver. The resulting asssembly is contained in 15,998 gapless contigs.

All components within this track are of fragment type "D": draft whole genome shotgun contig.
