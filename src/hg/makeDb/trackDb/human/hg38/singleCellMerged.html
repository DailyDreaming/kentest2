Description

This track displays single cell data from 12 papers covering 14 organs. Cells are grouped together by organ and cell type. The cell types are based on annotations published alongside the papers. These were curated at UCSC to as much as possible use the same cell type terminanologies across papers and organs. In some cases we merged together small populations of cells annotated as distict but related types into a single type so as to have enough cells to call gene expression levels accurate. The gene expression levels are normalized so that that total level of expression for all genes in a single cell or cell type adds up to one million.

Methods

Each organ (or tissue) was integreted and curated into the Genome Browser indiviually. (Make table of contributing tracks here). All components were normalized to be in parts per million using the matrixNormalize command available from UCSC. Metadata was cleaned up using the tabToTabDir tool. The major clean ups were unpacking abbreviations, replacing jargon with standard English, choosing shorted terms to shorten long labels, labeling outliers, etc. Before integration we invited the original data producers as well as local biologists and informaticions to view the data.
