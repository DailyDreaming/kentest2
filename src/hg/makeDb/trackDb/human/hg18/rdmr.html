Description

This track provides the location of genomic regions that show differential DNA methylation (DNAm) between induced pluripotent stem (iPS) cells and their parental fibroblasts. M values represent averaged methylation values across all samples in a given cell type. R-DMRs were identified using Comprehensive High-Throughput Arrays for Relative Methylation (CHARM) analysis. For a detailed description of CHARM analysis, please see Irizarry RA et al., 2008.
